# 職種マップ診断 RIASECプロトタイプ

## 使い方

1. `index.html` をブラウザで開く
2. 「診断を始める」を押す
3. 質問に答える
4. 結果画面でRIASECスコア、職種カード、大学中にやることを見る

## ファイル構成

- `index.html`：画面構造
- `style.css`：デザイン
- `script.js`：質問、分岐、スコア計算、結果表示

## 仕様

- RIASEC 6軸：R/I/A/S/E/C
- 共通質問12問
- 分岐質問6問
- 序盤の点数で「技術・研究」「対人・推進」「表現・管理」に分岐
- 上位2軸で結果タイプを判定
- 回答データは保存しない

## 次に追加するとよい機能
